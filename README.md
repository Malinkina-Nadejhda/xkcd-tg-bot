# `tg_xkcd_bot.py`

Телеграм бот для автоматической публикации комиксов XKCD.

**Как установить:**

- Должен быть установлен `Python` версии 3.13
- Клонируйте в корневую папку проекта imgs_uploader.py файл
- Клонируйте в корневую папку проекта xkcd.py файл.
- Используйте `pip` для установки зависимостей

```bash
pip install -r requirements.txt
```


## Настройка окружения

1. Получите токен для бота у официального бота [@BotFather](https://t.me/botfather).
2. Узнайте ID своего канала (например, с помощью бота [@GetMyIdBot](https://t.me/Getmyid_bot)).
3. Создайте файл `.env` в корне проекта по образцу файла `.env.example`.
4. Добавьте в `.env` ваши реальные данные:
   ```env
   TELEGRAM_TOKEN=ваш_токен_бота
   TG_CHAT_ID=ваш_id_канала
   PUBLIC_INTERVAL=14400
   ```
   *Параметр `PUBLIC_INTERVAL` задаёт частоту публикаций в секундах (по умолчанию 14400 секунд / 4 часа).*

## Запуск

Для работы бота доступны два режима. Вы можете использовать флаг `--help` для вывода подробной справки по аргументам.

### 1. Автоматическая публикация комиксов из сети (`auto`)

Бот будет сам выбирать случайный комикс XKCD, скачивать его, публиковать вместе с авторским комментарием в канал и удалять временный файл. Для настройки частоты публикаций используйте флаг `--interval` (время указывается в секундах). Если флаг не задан, бот возьмет значение `PUBLIC_INTERVAL` из файла `.env` (по умолчанию — 14400 секунд / 4 часа).

```bash
# Запуск с интервалом по умолчанию.
python .\tg_xkcd_bot.py --mode auto

# Запуск с заданным интервалом.
python .\tg_xkcd_bot.py --mode auto --interval 300
```

### 2. Публикация локального изображения (`img`)

Отправка одного конкретного изображения с вашего компьютера. Для этого режима обязательно укажите путь к файлу через флаг `--path`.

```bash
python .\tg_xkcd_bot.py --mode img --path "D:\1.jpg
```

## `xkcd.py`

Модуль для работы с API сайта xkcd. Получает номер последнего комикса, а также ссылки на картинки и авторские комментарии.

## `imgs_uploader.py`

Файловый менеджер. Отвечает за создание временных папок, скачивание картинок на компьютер и их последующее удаление.
